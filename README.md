# Zaibatsu TTRPG System

Цифровая реализация настольной ролевой игры Zaibatsu для платформы Foundry VTT.

## Описание

Zaibatsu - это научно-фантастическая ролевая игра в сеттинге киберпанка, где игроки выступают в роли агентов могущественных корпораций. Система использует d6 для механик и включает уникальные элементы прогрессии персонажей через ранги (от Агента до Начальника отдела).

## Особенности системы

- **Характеристики персонажа**: Сила, Ловкость, Выносливость, Интеллект, Образование, Социальное влияние
- **Система рангов**: 7 уровней карьерного роста с соответствующими бонусами
- **Управление инвентарем**: Система слотов с ограничениями по весу
- **Ретрогенные адаптации**: Уникальные способности персонажей
